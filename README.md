# build and install
- install Rust first
- git clone this repo
- cargo build --release
- cargo install --path .
- cd to any empty folder you like

# 操作步骤for alice

full-DH-with-sign alice

or

cargo run -- alice

或者

./target/debug/full-DH-with-sign alice
